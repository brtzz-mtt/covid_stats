BASH scripts

Bundled pytests can be run and resulting code-coverage reports saved under ./htmlcov as follows: ./toolbox.sh checkup

A static html structure of application can be esported under ./build as follows (includes checkup): ./toolbox.sh build

Additionaly, local dev branch (with appropriate privileges) can be merged in master like this: ./toolbox.sh push
